About OTD

One-Time Downloads

OTD is a lightweight, self-hosted tool for creating secure, one-time download links. Share files with anyone - once the link is used, it's gone. No accounts, no cloud storage, no tracking.

Built with Rust for performance and reliability. No JavaScript runtime, no container orchestration, no dependencies you don't need. Just a single binary that does one thing well.

OTD is free and open source. Use it, modify it, host it yourself.
